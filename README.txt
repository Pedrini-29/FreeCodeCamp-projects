This folder is a collection of the certification projects that I completed to obtain the FreeCodeCamp Relational Databases Course.

PostgresSQL was used to create the databases.
Bash was used for creating small applications that interacted with the database.

Project 1 - SALON: An application was designed to create an appointments scheduler for an hair saloon.
Project 2 - NUMBER GUESS: A random number generator and a guessing game - user's scores are saved into a database 
Project 4 - ELEMENTS: A database with the elements of the periodic table was created, an application guides you in the exploration of the periodic table
